# 标签合并 - 快速参考

## 一句话说明
将标签 A 的所有对话标记转移到标签 B，然后删除标签 A。

## 3 步完成合并

```
1️⃣  进入标签管理
   设置 → 标签管理 (http://localhost:8000/settings/tags)

2️⃣  选择目标标签
   找到要合并的标签 → 点击"编辑/停用" → 在底部"合并到另一个标签"区域选择目标

3️⃣  执行合并
   点击"合并到目标标签" → 确认 → 完成
```

## 常见用法

### 场景 1：去重
```
问题：有两个意思相同的标签
  - "未及时回复"
  - "回复不及时"

操作：将"未及时回复"合并到"回复不及时"

结果：所有标记都统一为"回复不及时"
```

### 场景 2：规范化
```
问题：标签命名不统一
  - "色差"
  - "做工问题"

操作：
  1. 创建统一标签"色差/做工问题"
  2. 将"色差"合并到"色差/做工问题"
  3. 将"做工问题"合并到"色差/做工问题"

结果：三个标签合并为一个标准标签
```
